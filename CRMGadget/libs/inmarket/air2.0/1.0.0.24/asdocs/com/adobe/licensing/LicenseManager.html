Classpublic class LicenseManager
InheritanceLicenseManager flash.events.EventDispatcher

The LicenseManager class is the application entry point for all licensing requests. It is used by an AIR application to check the licensing status. For base class documentation see the ActionScript 3.0 Reference for the Adobe Flash Platform.

ttl:Number  [read-only]

License time to live in units of milliseconds. This is the time remaining before the license is re-verified against the licensing server. When ttl = -1, the license does not need to be verified against the server.

checkLicense()method
public function checkLicense(owner:DisplayObject, applicationKey:String, checkUpdate:Boolean = false, debugMode:Boolean = false):void

checkLicense is the asynchronous method for checking the application's license.

Parameters

owner:DisplayObject — The main display object for the application.
 
applicationKey:String — A random number selected by the application developer. The number is immutable for the lifetime of the application. The number is in the form of a GUID. For example: BE46EE64-FFC5-AC6C-2A83-2123BAD9F60C.
 
checkUpdate:Boolean (default = false) — If true, the implementation should check for app updates and facilitate the app update process. If false, no update should be performed.
 
debugMode:Boolean (default = false) — This parameter is used to facilitate the licensing during the development of the application. When debugMode = true, the license provider should display a UI that allows the developer to "skip" the licensing check, or "reset/clear" the current license.checkLicense
