Licensing.swc

The Licensing.swc file is the library required to integrate licensing capabilities into your applications and content. It is located in the InMarket_sdk/air2.0/1.0.0.24/lib directory. For more information on how to add this SWC to your project see the documentation links below.

Documentation

The documentation consists of two components:

  • ActionScript API Reference for the LicenseManager class
  • Online documentation with code examples that explain how to integrate licensing capabilities into your applications
